How It Works

For Scorers: Open the scorer tool, optionally enter a tournament name (or leave blank for casual matches), enter team names and match settings, then start recording each ball. The app generates a unique Organization ID based on your tournament name to keep your matches organized. Share individual match links or the entire tournament link with spectators for live updates.

Tournament Management: When you enter a tournament name, it's saved to your device's history for quick autocomplete next time. Your tournament list is private and stored locally - other users won't see your tournaments in their dropdown.

For Viewers: Simply open the Live Score page - it automatically detects and displays the current live match. Or use the shared link (with org and match parameters) to watch a specific match across any device or browser.

For Everyone: Browse all matches on the Matches page, see which games are live with tournament badges (#TournamentName), and review completed match results with detailed statistics.

Technology

Built with modern web technologies including Firebase Realtime Database for instant synchronization and localStorage for privacy-focused tournament history. The app features real-time listeners that automatically update scores without page refreshes, organization-based database isolation for multi-user environments, smart match detection algorithms, tournament-based URL sharing with org and match parameters, and a responsive design that works beautifully on all screen sizes.

Privacy & Security

Your privacy and data integrity matter. Tournament names you create are stored only on your device using localStorage - they're never shared with other users. Each tournament gets a unique Organization ID that keeps your matches separate from others. Match scores are shared via Firebase for live viewing, but your personal tournament list remains private.

Edit Token Protection: Every match generates a secure edit token that controls who can modify scores. Share the public viewer link for spectators to watch safely (read-only mode), and share the private scorer link only with trusted co-scorers who need edit access. This prevents accidental changes and malicious tampering while keeping matches publicly viewable.
