Net Run Rate (NRR)

Automatically calculated using standard ICC method for League Match type only.

Formula

NRR = (Runs Scored ÷ Overs Faced) - (Runs Conceded ÷ Overs Bowled)

Key Points

  • NRR table appears only for League Match type
  • All-out innings use full overs allocation
  • Positive NRR = scoring faster than opponents
  • View detailed breakdown: click ⓘ icon on Matches page
