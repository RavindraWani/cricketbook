CricketBook is a free, real-time cricket scoring tool for local matches and tournaments. Score matches ball-by-ball and share live updates instantly with WhatsApp integration.

Core Features

  • Real-Time Scoring: Ball-by-ball tracking with instant updates across all devices
  • WhatsApp Sharing: One-tap sharing with custom messages - "No need to ask 'score kya hai?' every 5 minutes üòÑ"
  • Secure Access: Edit tokens protect scoring - public can watch, only authorized scorers can edit
  • Tournament Management: Organize matches with custom names and 8 match types (League, Semi-Final, Final, etc.)
  • Auto-Detection: Live Score page automatically finds and displays ongoing matches
  • Match History: Browse up to 20 recent matches with detailed over-by-over summaries
  • Flexible Settings: Configure overs limit (1-50), wickets (1-20 players), and team names
  • Privacy-First: Tournament history stored locally on your device only
  • No Sign-up: Start scoring immediately - 100% free forever
