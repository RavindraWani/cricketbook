🏏 About CricketBook

CricketBook is a free, real-time cricket scoring application designed for local cricket matches, tournaments, and friendly games. Our tool makes it easy to score matches ball-by-ball and share live updates instantly with players, friends, and family.

What We Offer

CricketBook provides three main ways to experience cricket matches:

📺 Live Score

Watch real-time cricket scores with automatic updates. The live score page automatically detects ongoing matches and displays ball-by-ball updates, over summaries, and match progress. Perfect for spectators who want to follow the action remotely.

📋 All Matches

Browse through match history with a comprehensive view of all matches - both completed and ongoing. See recent matches in an organized grid layout, track tournament progress, and quickly jump to any match for detailed viewing.

⚙️ Scorer Tool

A powerful scoring interface for match officials and scorers. Record every ball, wicket, extra, and milestone with precision. The scorer creates a unique match ID that can be shared for live viewing, making it easy for others to follow along in real-time.

Key Features

  • Real-time Synchronization: Scores update instantly across all devices using Firebase
  • Auto-Detection: Live Score page automatically finds and displays ongoing matches
  • Ball-by-Ball Tracking: Record every delivery with runs, wickets, and extras
  • Match History: View up to 20 recent matches in an organized grid
  • Live Badges: Clear indicators show which matches are currently in progress
  • Customizable Settings: Configure overs limit, wickets (supports 1-20 players), and team names
  • Share Viewer Links: Generate shareable links with mobile share options
  • Detailed Statistics: Track runs, wickets, extras, strike rates, and run rates
  • Match Summary: Over-by-over breakdown with complete ball-by-ball details
  • Mobile-First Design: Optimized for phones, tablets, and desktops
  • No Registration: Start using immediately without any sign-up
  • 100% Free: No hidden costs or premium features

How It Works

For Scorers: Open the scorer tool, enter team names and match settings, then start recording each ball. Share the viewer link with spectators for live updates.

For Viewers: Simply open the Live Score page - it automatically detects and displays the current live match. Or use the shared link to watch a specific match.

For Everyone: Browse all matches on the Matches page, see which games are live, and review completed match results.

Technology

Built with modern web technologies including Firebase Realtime Database for instant synchronization. The app features real-time listeners that automatically update scores without page refreshes, smart match detection algorithms, and a responsive design that works beautifully on all screen sizes.
