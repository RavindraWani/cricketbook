🏏 About CricketBook

Free, real-time cricket scoring platform for local matches. Create Quick Matches (single) or organize Tournaments (multiple matches) with password protection and automatic points tables.

Key Features

  • Quick Match - Single match without setup, instantly tracked in your browser
  • Real-time ball-by-ball scoring with instant Firebase sync
  • Two share links: View Link (watch live) & Editor Link (co-scoring)
  • WhatsApp sharing - Share live scores directly with one tap 📱
  • Global Match Browser - View all ongoing matches worldwide 🌍
  • Tournament mode - Group multiple matches with passwords & points tables
  • Automatic points table calculation for League matches
  • 8 match types: Friendly, League, Semi-Final, Final, and more
  • Flexible settings: 1-50 overs, 1-20 wickets per innings
  • Saved Matches - Resume incomplete matches anytime
  • Undo button - Reverse any action (unlimited undo history)
  • 100% free, no signup required, always free

How It Works

Quick Match: Single match without tournament - automatically saved to your device
Score: Record each ball with real-time Firebase sync across all viewers
Share: View Link (anyone can watch) or Editor Link (authorized scorers only)
Tournament: Multiple matches grouped together with password protection and automatic points tables
Resume: All incomplete matches auto-saved and accessible from home page

Net Run Rate (NRR)

Automatically calculated using standard ICC method for League Match type only.

Formula

NRR = (Runs Scored ÷ Overs Faced) - (Runs Conceded ÷ Overs Bowled)

Key Points

  • NRR table appears only for League Match type
  • All-out innings use full overs allocation for calculation
  • Positive NRR = scoring faster than opponents
  • Negative NRR = conceding runs faster than scoring
  • View detailed breakdown: Click ⓘ icon on Matches page
